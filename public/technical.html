PHẦN 1: Giới thiệu

1. Firebase là gì?

Firebase là một nền tảng do Google cung cấp hoạt động trên cơ sở đám mây - Cloud để phát triển ứng dụng di động và trang web, bao gồm các API đơn giản và mạnh mẽ mà không cần backend hay server.


2. Tên miền là gì?

Tên miền là tên của một website hoạt động trên internet, đóng vai trò là một địa chỉ tĩnh, cố định. Nó giống như là địa chỉ nhà hay mã zip code để giúp các thiết bị định tuyến vệ tinh dẫn đường cho hàng hóa lưu thông, một trình duyện cũng cần một tên miền để dẫn đường tới nơi chứa website của bạn (web server).

PHẦN 2: Tạo Project Firebase

1. Vào trang web firebase.google.com



2. Tiến hành đăng nhập vào firebase bằng tài khoản google.

3. Sau khi đăng nhập chọn Get start.



4. Tạo một dự án mới.



5. Đặt tên cho dự án.



6. Chọn Carry on.



7. Chọn Default Account for Firebase.



8. Chọn vào đây để tạo website cho dự án của bạn.



9. Đặt tên cho website của bạn và chọn đăng ký.



10. Vậy là đã đăng ký thành công một web app trên firebase để chứa dự án

PHẦN 3: Deploy Firebase

Bước 1: Để deploy firebase, trước tiên cần cài đặt Node Js. Link cài đặt: https://nodejs.org/en/



Bước 2:
1. Mở command line bằng tổ hợp phím Window + R.

2. Nhập lệnh “npm install -g firebase-tools”



Bước 3:
1. Tiếp theo cần tạo một thư mục trên máy tính để chưa dự án.



2. Vào thư mục vừa tạo và mở cmd.

3. Nhập lệnh “firebase login” để đăng nhập vào firebase.

4. Tiếp tục gõ lệnh “ firebase init hosting”


5. Chọn “Use an existing project” và chọn tên dự án bạn đã tạo ở trên.



6. Trả lời các câu hỏi và chờ hiện ra “Firebase initialization complete!”



7. Tiếp theo cần gõ lệnh “firebase deploy” để đưa code lên hosting của firebase là xong.



8. Truy cập vào đường link trong mục "Project console" hoặc "Hosting URL" để vào website của bạn.
